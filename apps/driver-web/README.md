# Mishwari Driver Web

Driver-facing web application for trip management and fleet operations.

## Getting Started

```bash
# Install dependencies (from monorepo root)
npm install

# Run development server
npm run dev

# Build for production
npm run build
npm run start
```

Open [http://localhost:3001](http://localhost:3001)

## Features

- Fleet Management
- Trip Creation (Scheduled & Flexible)
- Physical Booking Recording
- Trip & Booking Management

## Tech Stack

- Next.js 14 (Pages Router)
- TypeScript
- Tailwind CSS
- Redux Toolkit
- Shared packages from monorepo
